# 🔥 FastFurnace

**FastFurnace** is a lightweight and fully configurable Paper plugin that speeds up furnaces on your Minecraft server.

Perfect for survival servers, SMPs, or custom setups where you want faster smelting without breaking the game balance.

---

## ⚡ Features

- 🔥 Increase furnace smelting speed
- 🍖 Works with all furnace types:
  - Furnace
  - Blast Furnace
  - Smoker
- ⚙️ Fully configurable
- 🚀 Lightweight & optimized (no lag)
- 📦 Easy plug-and-play setup

---
